Icons for SenangStart.

A professional, open-source icon set designed for clarity and simplicity.

Quick Installation

Choose how you want to include SenangStart Icons in your project. Use NPM for module bundlers or the CDN for quick prototyping.

Install via NPM 
npm install senangstart-icons
CDN Link Recommended for static sites 
<link rel="stylesheet" href="https://cdn.example.com/senangstart.min.css" />
<script src="https://cdn.example.com/senangstart-icon.min.js"></script>

How to use

Simply use the ss-icon custom element and pass the icon name to the icon attribute.

Size: Default
Size: 48px 
<!-- Basic Usage -->
<ss-icon icon="home"></ss-icon>

<!-- With Tailwind Classes -->
<ss-icon icon="heart" class="text-brand-600 w-12 h-12"></ss-icon>
